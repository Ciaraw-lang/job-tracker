Description:
A web application called Smart Job Tracker was created to assist users in managing and organizing their job search process in one location. 
Users can log job applications, keep an eye on application statuses, and compare resumes to job descriptions to find keywords and abilities that are lacking. 
The initiative aims to streamline the application process and assist users in maintaining organization during their job hunt.

Features:
Add, modify, and remove job applications
Monitor the status of your applications (Applied, Interview, Offer, Rejected).
Keep track of job titles, application dates, and business names.
Analysis of resumes using job descriptions
Matching keywords and skills
User-friendly and responsive interface
A well-organized application management dashboard

Technologies used:
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB

Installation: 
Clone the repository: git clone YOUR_REPOSITORY_LINK
Install Dependencies: npm install
Start the server:node server.js

Run the application: 
Open browser and type: http://localhost:3000

How it works:
The dashboard is used by users to submit and manage job applications.
The database contains application data.
Users can compare their resumes with job descriptions by uploading or pasting their resumes.
In order to make resumes better, the algorithm finds matched and missing keywords.
Throughout the job search process, users are able to track their progress.
